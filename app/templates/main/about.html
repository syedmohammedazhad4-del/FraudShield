About This System

The Insurance Fraud Detection System is a machine learning powered web application designed to identify potentially fraudulent insurance claims. By analyzing claim attributes and patterns, the system provides real-time fraud predictions with confidence scoring.

How It Works
1. Data Input

Users enter insurance claim details through a guided multi-step wizard form with 25 key attributes.

2. Encoding

Categorical features are label-encoded and numerical features are processed to match the training data format.

3. ML Model

The encoded data is fed into a trained machine learning model that evaluates fraud probability.

4. Prediction

The system returns a Fraud or Legitimate verdict along with a confidence score and detailed analysis.

Technology Stack
Flask

Python web framework powering the backend API and routing

scikit-learn

Machine learning library for model training and predictions

Bootstrap 5

Responsive front-end framework for modern UI design

Chart.js

JavaScript library for interactive data visualizations
